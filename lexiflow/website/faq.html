Utilisation Générale

Comment installer LexiFlow sur Chrome ? +

L'installation de LexiFlow est simple et rapide :

  • Visitez le Chrome Web Store
  • Cliquez sur "Ajouter à Chrome"
  • Confirmez en cliquant sur "Ajouter l'extension"
  • L'icône LexiFlow 🌐 apparaîtra dans votre barre d'outils

L'extension est immédiatement active après l'installation. Aucune configuration supplémentaire n'est nécessaire pour commencer !

Quelles langues sont supportées par LexiFlow ? +

LexiFlow supporte actuellement 11 langues majeures :

  • 🇫🇷 Français
  • 🇺🇸 Anglais
  • 🇸🇦 Arabe
  • 🇪🇸 Espagnol
  • 🇩🇪 Allemand
  • 🇮🇹 Italien
  • 🇵🇹 Portugais
  • 🇷🇺 Russe
  • 🇯🇵 Japonais
  • 🇰🇷 Coréen
  • 🇨🇳 Chinois

La traduction fonctionne dans les deux sens entre toutes ces langues. La détection automatique identifie la langue source pour vous.

Comment utiliser les flashcards pour mémoriser ? +

Les flashcards sont un outil puissant pour mémoriser le vocabulaire :

  • Après chaque traduction, cliquez sur le bouton "💾 Flashcard"
  • La carte est automatiquement créée avec le mot original et sa traduction
  • Accédez à vos flashcards via l'icône de l'extension
  • Organisez-les par langues ou thèmes
  • Révisez régulièrement pour une mémorisation optimale

Version gratuite : 50 flashcards | Version Premium : 200 flashcards

Quelle est la différence entre les limites de caractères ? +

Les limites de caractères déterminent la longueur maximale du texte à traduire :

  • Version Gratuite : 100 caractères - Parfait pour les mots et phrases courtes
  • Version Premium : 350 caractères - Idéal pour les paragraphes et textes complexes

Cette limite s'applique à chaque traduction individuelle. Le nombre de traductions par jour est illimité dans les deux versions.

LexiFlow fonctionne-t-il sur tous les sites web ? +

LexiFlow fonctionne sur la grande majorité des sites web. Cependant, certaines exceptions existent :

  • Sites bancaires et financiers (pour des raisons de sécurité)
  • Pages Chrome internes (chrome://)
  • Chrome Web Store
  • Certains sites avec des restrictions de sécurité strictes

Sur tous les autres sites, l'extension fonctionne parfaitement : actualités, réseaux sociaux, blogs, e-commerce, etc.

Comment désactiver temporairement LexiFlow ? +

Vous avez plusieurs options pour désactiver LexiFlow :

  • Désactivation temporaire : Cliquez sur l'icône LexiFlow et utilisez le toggle on/off
  • Pour un site spécifique : Utilisez le menu de l'extension pour exclure le site actuel
  • Désactivation complète : Via le gestionnaire d'extensions Chrome

L'extension garde vos paramètres et flashcards même désactivée.

Puis-je utiliser LexiFlow hors ligne ? +

LexiFlow nécessite une connexion internet active pour les traductions. Cependant :

  • Vos flashcards sont stockées localement et accessibles hors ligne
  • L'historique des traductions reste consultable sans connexion
  • Les paramètres sont sauvegardés localement

Avec Premium, la synchronisation cloud se fait automatiquement dès que vous êtes reconnecté.

Comment utiliser le raccourci clavier pour traduire ? +

Le raccourci clavier permet une traduction ultra-rapide :

  • Windows/Linux : Sélectionnez le texte puis appuyez sur Ctrl + Q
  • Mac : Sélectionnez le texte puis appuyez sur Cmd + Q

Vous pouvez personnaliser ce raccourci dans les paramètres de l'extension ou le désactiver si vous préférez utiliser uniquement le bouton visuel.

Problèmes Techniques

Que faire si l'extension ne fonctionne pas sur un site ? +

Si LexiFlow ne fonctionne pas sur un site spécifique :

  • Rafraîchissez la page (Ctrl/Cmd + R)
  • Vérifiez que l'extension est activée via l'icône dans la barre d'outils
  • Désactivez temporairement les autres extensions (conflits possibles)
  • Videz le cache de votre navigateur
  • Réinstallez LexiFlow si le problème persiste

Si le problème continue, contactez notre support avec l'URL du site concerné.

Comment signaler un bug ou un problème ? +

Nous prenons tous les bugs au sérieux. Pour nous aider :

  • Utilisez le formulaire de contact
  • Ou envoyez un email à support@lexiflow.com
  • Incluez : URL du site, description du problème, captures d'écran
  • Mentionnez votre version de Chrome et système d'exploitation

Réponse garantie sous 48h (24h pour les utilisateurs Premium).

L'extension ralentit-elle mon navigateur ? +

LexiFlow est optimisé pour une performance maximale :

  • Utilisation mémoire minimale (< 50MB)
  • Activation uniquement lors de la sélection de texte
  • Pas de scripts en arrière-plan inutiles
  • Cache intelligent pour les traductions fréquentes
  • Code optimisé et régulièrement mis à jour

Si vous constatez un ralentissement, vérifiez d'abord les autres extensions installées.

Pourquoi certaines traductions semblent incorrectes ? +

La qualité de traduction dépend de plusieurs facteurs :

  • Version gratuite : Utilise Google Translate (bon mais basique)
  • Mots ambigus : Peuvent avoir plusieurs sens selon le contexte
  • Expressions idiomatiques : Difficiles à traduire littéralement
  • Termes techniques : Nécessitent un contexte spécialisé

Solution : DeepSeek AI Premium résout 95% de ces problèmes grâce à l'analyse contextuelle avancée.

Comment mettre à jour LexiFlow ? +

Les mises à jour sont automatiques via Chrome :

  • Chrome vérifie les mises à jour toutes les quelques heures
  • Installation automatique en arrière-plan
  • Aucune action requise de votre part
  • Vos données et paramètres sont préservés

Pour forcer une mise à jour : chrome://extensions → Activer le mode développeur → Mettre à jour.

Que faire si mes flashcards ont disparu ? +

Ne paniquez pas ! Vos flashcards peuvent être récupérées :

  • Vérifiez d'abord que vous êtes connecté au bon compte
  • Rafraîchissez l'extension (désactiver/réactiver)
  • Version Premium : La sync cloud les restaurera automatiquement
  • Version gratuite : Vérifiez le stockage local de Chrome

Si le problème persiste, notre support technique peut vous aider à récupérer vos données.

Confidentialité et Données

Mes traductions sont-elles stockées ou partagées ? +

Votre confidentialité est notre priorité :

  • Les traductions ne sont PAS stockées sur nos serveurs
  • Aucun partage avec des tiers
  • Traitement local sur votre appareil
  • Historique stocké uniquement en local (ou cloud chiffré pour Premium)
  • Vous pouvez effacer vos données à tout moment

Consultez notre politique de confidentialité complète pour plus de détails.

LexiFlow collecte-t-il mes données personnelles ? +

Nous collectons le minimum de données nécessaires :

  • Version gratuite : Aucune donnée personnelle collectée
  • Version Premium : Email et infos de paiement uniquement
  • Pas de tracking comportemental
  • Pas d'analyse de vos sites visités
  • Pas de vente de données

Nous respectons strictement le RGPD et les lois de protection des données.

Où sont stockées mes flashcards ? +

Le stockage dépend de votre version :

  • Version gratuite : Stockage local dans votre navigateur
  • Version Premium : Stockage local + cloud sécurisé
  • Chiffrement de bout en bout pour le cloud
  • Serveurs sécurisés en Europe (RGPD)
  • Backup automatique quotidien (Premium)

Vous gardez le contrôle total : export et suppression possibles à tout moment.

Comment supprimer toutes mes données ? +

Vous pouvez supprimer vos données à tout moment :

  • Dans l'extension : Paramètres → Effacer toutes les données
  • Compte Premium : Dashboard → Supprimer le compte
  • Suppression immédiate et définitive
  • Aucune conservation après suppression
  • Confirmation par email envoyée

Note : Cette action est irréversible. Exportez vos données avant si nécessaire.

LexiFlow utilise-t-il des cookies ? +

Notre utilisation des cookies est minimale et transparente :

  • Extension : Aucun cookie utilisé
  • Site web : Cookies essentiels uniquement
  • Pas de cookies de tracking publicitaire
  • Pas de cookies tiers (sauf Stripe pour les paiements)
  • Respect total de votre vie privée

Vous pouvez bloquer les cookies sans affecter le fonctionnement de l'extension.
