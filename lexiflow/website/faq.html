How do I use flashcards to memorize? +

Flashcards are a powerful tool for memorizing vocabulary:

  • After each translation, click the "üíæ Flashcard" button

  • Access your flashcards via the extension icon
  • Organize them by languages or themes

What's the difference between character limits? +

Character limits determine the maximum length of text to translate:

  • Free Version: 150 characters - Perfect for words and short phrases
  • Premium Version: unlimited characters - Ideal for paragraphs and complex texts

This limit applies to each individual translation. The number of daily translations is unlimited in both versions.

Does LexiFlow work on all websites? +

LexiFlow works on the vast majority of websites. However, some exceptions exist:

  • Banking and financial sites (for security reasons)
  • Internal Chrome pages (chrome://)
  • Chrome Web Store
  • Some sites with strict security restrictions

On all other sites, the extension works perfectly: news, social media, blogs, e-commerce, etc.

How do I temporarily disable LexiFlow? +

You have several options to disable LexiFlow:

  • Temporary disable: Click the LexiFlow icon and use the on/off toggle
  • For a specific site: Use the extension menu to exclude the current site
  • Complete disable: Via Chrome's extension manager

The extension keeps your settings and flashcards even when disabled.

Can I use LexiFlow offline? +

LexiFlow requires an active internet connection for translations. However:

  • Your flashcards are stored locally and accessible offline
  • Translation history remains available without connection
  • Settings are saved locally

With Premium, cloud synchronization happens automatically when you're back online.

How do I use the keyboard shortcut to translate? +

The keyboard shortcut allows ultra-fast translation:

  • Windows/Linux: Select text then press Ctrl + Q
  • Mac: Select text then press Cmd + Q

You can customize this shortcut in extension settings or disable it if you prefer using only the visual button.

Technical Issues

What if the extension doesn't work on a site? +

If LexiFlow doesn't work on a specific site:

  • Refresh the page (Ctrl/Cmd + R)
  • Check the extension is enabled via the icon in toolbar
  • Temporarily disable other extensions (possible conflicts)
  • Clear your browser cache
  • Reinstall LexiFlow if problem persists

If the problem continues, contact our support with the site URL.

How do I report a bug or issue? +

We take all bugs seriously. To help us:

  • Use the contact form
  • Or email support@lexiflow.com
  • Include: Site URL, problem description, screenshots
  • Mention your Chrome version and operating system

Guaranteed response within 48h (24h for Premium users).

Does the extension slow down my browser? +

LexiFlow is optimized for maximum performance:

  • Minimal memory usage (< 50MB)
  • Activation only when selecting text
  • No unnecessary background scripts
  • Smart cache for frequent translations
  • Optimized and regularly updated code

If you notice slowdown, first check other installed extensions.

Why do some translations seem incorrect? +

Translation quality depends on several factors:

  • Free version: Uses Google Translate (good but basic)
  • Ambiguous words: Can have multiple meanings depending on context
  • Idiomatic expressions: Difficult to translate literally
  • Technical terms: Need specialized context

Solution: Premium DeepSeek AI solves 95% of these issues with advanced contextual analysis.
