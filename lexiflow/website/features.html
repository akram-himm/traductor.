Questions Fréquentes

Comment activer l'apparition automatique du bouton ? +

Le bouton de traduction apparaît automatiquement dès que l'extension est installée et activée. Vous pouvez désactiver cette fonction dans les paramètres si vous préférez utiliser le raccourci clavier Ctrl+Q (ou Cmd+Q sur Mac).

Quelle est la différence entre les services de traduction ? +

La version gratuite utilise Google Translate et MyMemory pour des traductions rapides et efficaces. La version Premium inclut DeepSeek AI qui offre une compréhension contextuelle avancée, parfaite pour les textes complexes, les expressions idiomatiques et les nuances linguistiques subtiles.

Puis-je utiliser LexiFlow hors ligne ? +

LexiFlow nécessite une connexion internet pour les traductions en temps réel. Cependant, vos flashcards créées sont stockées localement et peuvent être consultées hors ligne. La synchronisation cloud (Premium) se fait automatiquement dès que vous êtes reconnecté.

Comment fonctionne le Mode Pratique Premium ? +

Le Mode Pratique utilise l'algorithme de révision espacée scientifiquement prouvé. Il vous propose des quiz personnalisés basés sur vos flashcards, ajuste automatiquement la fréquence de révision selon vos performances, et fournit des statistiques détaillées pour optimiser votre apprentissage.

LexiFlow fonctionne-t-il sur tous les sites web ? +

LexiFlow fonctionne sur la grande majorité des sites web. Certains sites avec des restrictions de sécurité strictes (comme les sites bancaires) peuvent limiter son fonctionnement. L'extension respecte toujours votre vie privée et ne collecte aucune donnée personnelle.
