Frequently Asked Questions

How do I enable automatic button appearance? +

The translation button appears automatically as soon as the extension is installed and activated. You can disable this feature in settings if you prefer to use only the keyboard shortcut Ctrl+Q (or Cmd+Q on Mac).

What's the difference between translation services? +

The free version uses Google Translate and MyMemory for fast and efficient translations. The Premium version includes DeepSeek AI which offers advanced contextual understanding, perfect for complex texts, idiomatic expressions and subtle linguistic nuances.

Can I use LexiFlow offline? +

LexiFlow requires an internet connection for real-time translations. However, your created flashcards are stored locally and can be accessed offline. Cloud synchronization (Premium) happens automatically when you're back online.

How does Premium Practice Mode work? +

Practice Mode uses the scientifically proven spaced repetition algorithm. It offers personalized quizzes based on your flashcards, automatically adjusts review frequency according to your performance, and provides detailed statistics to optimize your learning.

Does LexiFlow work on all websites? +

LexiFlow works on the vast majority of websites. Some sites with strict security restrictions (like banking sites) may limit its functionality. The extension always respects your privacy and collects no personal data.
